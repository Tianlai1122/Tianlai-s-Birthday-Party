# 🎉 生日派对网站

一个高级、简约的生日派对互动网站，具有实时数据保存和后台监控功能。

## ✨ 功能特点

### 🎨 前端功能
- **倒计时**：实时显示距离派对开始的时间
- **派对信息**：时间、地点、着装、联系方式
- **时间安排**：详细的派对流程时间表
- **团队介绍**：DJ、大厨、调酒师等所有工作人员
- **互动功能**：
  - 🐱 小馋猫列表：报名想吃夜宵的朋友
  - 🍺 逃酒排行榜：记录每个人要喝的杯数
  - 🎮 游戏偏好投票：统计大家喜欢的游戏类型
  - 👍 给提拉米苏大师点赞
- **游戏介绍**：各种派对游戏和教程视频链接

### 💾 后端功能
- **数据持久化**：所有用户输入自动保存
- **访问统计**：记录访问次数和访问历史
- **后台监控**：实时查看所有数据统计
- **数据导出**：可导出所有数据为JSON文件

### 🎨 设计特点
- 高级简约的现代设计
- 渐变色彩和流畅动画
- 完全响应式，支持手机和电脑
- Party风格的视觉效果

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动服务器

```bash
npm start
```

或者使用开发模式（自动重启）：

```bash
npm run dev
```

### 访问网站

打开浏览器访问：`http://localhost:3000`

## 🎮 使用说明

### 用户功能

1. **加入小馋猫列表**
   - 输入名字
   - 点击"我是小馋猫"按钮

2. **加入逃酒排行榜**
   - 输入名字
   - 输入要逃的杯数
   - 点击"加入排行榜"按钮
   - 显示格式会自动转换为"今晚我要喝X杯"

3. **提交游戏偏好**
   - 输入名字
   - 选择喜欢的游戏类型
   - 点击"提交偏好"按钮

4. **给Krystal点赞**
   - 直接点击"给她点赞"按钮

### 管理员功能

点击右下角的 📊 按钮打开管理员面板，可以：

- 查看访问统计
- 查看所有互动数据
- 导出数据为JSON文件
- 清空所有数据（谨慎使用）

## 🔧 API 接口

### 获取数据
```
GET /api/data
```

### 保存数据
```
POST /api/data
Body: { foodies, drinkers, gamePreferences, krystalLikes, visits }
```

### 记录访问
```
POST /api/visit
```

### 获取统计信息
```
GET /api/stats
```

### 管理员接口
```
GET  /api/admin/full-data    # 获取完整数据
GET  /api/admin/export       # 导出数据
POST /api/admin/clear        # 清空数据
```

## 💡 技术栈

- **前端**：原生 HTML + CSS + JavaScript
- **后端**：Node.js + Express
- **数据存储**：JSON 文件 + localStorage（双重保存）
- **样式**：CSS3 动画 + 渐变 + 响应式设计

## 🎨 自定义

### 修改派对时间
编辑 `app.js` 中的：
```javascript
const partyDate = new Date('2024-11-22T20:30:00');
```

### 修改颜色主题
编辑 `styles.css` 中的 CSS 变量：
```css
:root {
    --primary: #ff6b9d;
    --secondary: #c06c84;
    --accent: #ffa07a;
    --dark: #1a1a2e;
    --light: #f8f9fa;
}
```

### 修改服务器端口
编辑 `server.js` 中的：
```javascript
const PORT = 3000;
```

## 📱 响应式设计
