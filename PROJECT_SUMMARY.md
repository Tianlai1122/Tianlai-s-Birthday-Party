## 🎯 核心特性

### 1. 双重数据保存
```
用户操作
    ↓
localStorage（浏览器）
    ↓
服务器 API
    ↓
party-data.json（文件）
```

**优点：**
- 即使服务器断开，数据也不会丢失
- 浏览器刷新后数据仍然存在
- 服务器重启后自动恢复数据

### 2. 实时倒计时
```javascript
派对时间：2024-11-22 20:30:00
实时显示：X天 X小时 X分钟 X秒
派对开始后：🎉 派对进行中！
```

### 3. 智能排序
- 逃酒排行榜：按杯数从多到少
- 游戏偏好：按投票人数排序
- 前三名显示奖牌 🥇🥈🥉

### 4. 数据可视化
- 柱状图展示游戏偏好
- 百分比宽度动态计算
- 颜色渐变美化

## 🚀 部署建议

### 局域网部署
