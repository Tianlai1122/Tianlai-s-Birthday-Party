# 🚀 快速启动指南

## 第一步：安装 Node.js

如果还没有安装 Node.js，请访问 [nodejs.org](https://nodejs.org/) 下载并安装。

验证安装：
```bash
node --version
npm --version
```

## 第二步：安装依赖

在项目文件夹中打开终端，运行：

```bash
npm install
```

这会安装所有必要的依赖包（express 和 cors）。

## 第三步：启动服务器

```bash
npm start
```

你会看到类似这样的输出：

```
╔════════════════════════════════════════╗
║   🎉 生日派对服务器已启动！            ║
║                                        ║
║   访问地址: http://localhost:3000      ║
║   API地址: http://localhost:3000/api  ║
║                                        ║
╚════════════════════════════════════════╝
```

## 第四步：访问网站

打开浏览器，访问：

- **主页面**：http://localhost:3000
- **管理员后台**：http://localhost:3000/admin.html

## 🎯 主要功能

### 用户端（主页面）

1. **查看派对信息**
   - 时间、地点、着装要求
   - 时间安排表
   - 团队成员介绍

2. **互动功能**
   - 🐱 加入小馋猫列表
   - 🍺 加入逃酒排行榜
   - 🎮 提交游戏偏好
   - 👍 给 Krystal 点赞

3. **查看游戏**
   - 各种派对游戏介绍
   - 游戏教程视频链接

### 管理员端（后台页面）

1. **实时监控**
   - 访问统计
   - 所有互动数据
   - 自动每5秒刷新

2. **数据管理**
   - 导出数据为 JSON
   - 清空所有数据
   - 查看访问历史

## 📱 移动端访问

### 在同一 WiFi 网络下

1. 找到你的电脑 IP 地址：

**Mac/Linux:**
```bash
ifconfig | grep "inet "
```

**Windows:**
```bash
ipconfig
```

2. 在手机浏览器访问：
```
http://你的IP地址:3000
```

例如：`http://192.168.1.100:3000`

## 🔧 常见问题

### 端口被占用

如果 3000 端口被占用，编辑 `server.js`：

```javascript
const PORT = 3001; // 改成其他端口
```

### 数据丢失

数据保存在两个地方：
1. `party-data.json` 文件（服务器端）
2. 浏览器 localStorage（客户端）

建议定期从管理员后台导出数据备份。

### 无法访问

1. 确保服务器正在运行
2. 检查防火墙设置
3. 确保使用正确的 IP 地址和端口

## 🛑 停止服务器

在终端按 `Ctrl + C` 停止服务器。

数据会自动保存到 `party-data.json` 文件。

## 💡 小技巧

1. **自动重启**（开发模式）
   ```bash
   npm run dev
   ```
   修改代码后服务器会自动重启

2. **后台运行**（Mac/Linux）
   ```bash
   nohup npm start &
   ```

3. **查看实时日志**
   服务器会在终端显示所有请求日志

## 🎨 自定义

### 修改派对时间

编辑 `app.js` 第 20 行：
```javascript
const partyDate = new Date('2024-11-22T20:30:00');
```

### 修改颜色主题

编辑 `styles.css` 第 9-15 行：
```css
:root {
    --primary: #ff6b9d;
    --secondary: #c06c84;
    --accent: #ffa07a;
    /* ... */
}
```

### 修改服务器端口

编辑 `server.js` 第 6 行：
```javascript
const PORT = 3000;
```

同时修改 `app.js` 第 2 行：
```javascript
const API_URL = 'http://localhost:3000/api';
```
